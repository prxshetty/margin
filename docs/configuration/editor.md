# Editor Settings

Customize the writing surface.

## Document Outline

Toggle the interactive structure outline on the left side of the editor. The outline shows headings from your document and lets you click to jump between sections.

## Editor Statistics

Display word count and/or character count at the bottom of the editor:

- **Words & Characters** (default)
- **Words Only**
- **Characters Only**
- **None**

## Text Style

Change the typography of the editor. All styles use system fonts -- nothing is downloaded.

| Style | Font Stack | Description |
|-------|-----------|-------------|
| **System** | System sans-serif | Neutral app-native text for everyday drafting. |
| **Editorial** | Georgia, Times New Roman | Serif document text with a magazine-like rhythm. |
| **Manuscript** | Charter, Iowan Old Style, Georgia | Roomier serif text for chapter work and revision. |
| **Technical** | SF Pro Text, system sans-serif | Sharper spacing and monospace-friendly code blocks. |
| **Warm Sans** | Avenir Next, Trebuchet MS, system sans-serif | Softer humanist sans text without getting decorative. |
