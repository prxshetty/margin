Writing Guide

On this page

Editor Settings ​

Customize the writing surface.

Document Outline ​

Toggle the interactive structure outline on the left side of the editor. The outline shows headings from your document and lets you click to jump between sections.

Editor Statistics ​

Display word count and/or character count at the bottom of the editor:

  • Words & Characters (default)
  • Words Only
  • Characters Only
  • None

Text Style ​

Change the typography of the editor. All styles use system fonts -- nothing is downloaded.

StyleFont StackDescription
SystemSystem sans-serifNeutral app-native text for everyday drafting.
EditorialGeorgia, Times New RomanSerif document text with a magazine-like rhythm.
ManuscriptCharter, Iowan Old Style, GeorgiaRoomier serif text for chapter work and revision.
TechnicalSF Pro Text, system sans-serifSharper spacing and monospace-friendly code blocks.
Warm SansAvenir Next, Trebuchet MS, system sans-serifSofter humanist sans text without getting decorative.
